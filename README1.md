# Modelo de Fabricación Distribuida CNC (OTC Getafe)

Este proyecto desarrolla un modelo avanzado de Fabricación Distribuida para optimizar una red de seis talleres de mecanizado CNC mediante una Oficina Técnica Centralizada (OTC).

## 📝 Descripción
Desarrollo de un modelo de Fabricación Distribuida para optimizar una red de 6 talleres CNC mediante una Oficina Técnica Centralizada (OTC). Basado en un ecosistema CAD-CAM-Cloud y gemelos digitales, unifica la programación y garantiza la intercambiabilidad de nodos. Incluye App de gestión operativa en AppSheet y base de datos en la nube.

## 🛠️ Tecnologías Utilizadas
* **Base de Datos:** Google Sheets / Excel Cloud.
* **Interfaz de Gestión:** AppSheet (Ecosistema Cloud).
* **Metodología:** Gestión Holónica y Fabricación Distribuida.
* **Flujo de Trabajo:** Integración CAD-CAM-Cloud.

## 📁 Estructura del Repositorio
* `/Data`: Contiene la estructura de la base de datos (Excel) con el histórico de pedidos y configuración de nodos.
* `/App`: Referencias y documentación de la lógica interna de la aplicación.
