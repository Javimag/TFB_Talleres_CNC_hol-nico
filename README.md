# TFB_Talleres_CNC_hol-nico
Desarrollo de un modelo de Fabricación Distribuida para optimizar una red de 6 talleres CNC mediante una Oficina Técnica Centralizada (OTC). Basado en un ecosistema CAD-CAM-Cloud y gemelos digitales, unifica la programación y garantiza la intercambiabilidad de nodos. Incluye App de gestión operativa en AppSheet y base de datos en la nube.
